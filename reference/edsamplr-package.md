# edsamplr: First Semester Statistics Course Data Simulator

A tool for statistics educators to generate simulated data for all the
major concepts in a first-semester statistics course. Functions include
generating K groups/samples for binary, categorical, and quantitative
data, matched pairs quantitative data with specified parameters and/or
moments. There is also a function for correlated quantitative data with
specified moments, slope, and correlation coefficient. The quantitative
variables are produced using Fleishman's power transformation method.
You provide the parameters, and we provide the data.
